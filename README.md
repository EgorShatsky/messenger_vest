
# Кроссплатформенный мессенджер "Весть" 
![Logo](https://i.postimg.cc/259VL3X1/Vest.png)
# 
Проект разработан в рамках выпускной квалификационной работы. Приложение запускается на OC Windows, Linux и Android и имеет следующую функциональность:

    1. Регистрация и авторизация пользователей
    2. Подтверждение номера телефона через SMS
    3. Написание и отправка текстовых сообщений
    4. Отображение чатов и историй сообщений
    5. Настройка профиля пользователя
